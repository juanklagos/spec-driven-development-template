# SDD Consent / Consentimiento SDD

EN:
- Before creating/changing specs or implementation tasks, record explicit user consent.
- Command:
  `./scripts/confirm-user-consent.sh "User approved scope X"`

ES:
- Antes de crear/cambiar specs o tareas de implementación, registra consentimiento explícito del usuario.
- Comando:
  `./scripts/confirm-user-consent.sh "Usuario aprobó alcance X"`

Required log file / Archivo obligatorio:
- `.sdd/user-consent.log`
