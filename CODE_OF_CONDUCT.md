# Código de conducta

Este proyecto promueve colaboración respetuosa.

## Comportamiento esperado

- Respeto en toda interacción.
- Crítica técnica con claridad y sin ataques personales.
- Lenguaje comprensible para personas de distintos niveles.

## Comportamiento no aceptado

- Acoso.
- Discriminación.
- Ataques personales.

## Aplicación

La persona mantenedora del repositorio puede moderar, editar o rechazar contribuciones que rompan este código.
