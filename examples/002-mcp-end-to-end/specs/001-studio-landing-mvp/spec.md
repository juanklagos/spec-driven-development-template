# Spec 001: Studio Landing MVP

## Summary

Create the first public landing page for a small studio with service explanation, trust signals, and lead capture.

## Scope

- Hero section
- Services section
- Testimonials section
- Contact form
- Mobile-ready layout

## Acceptance criteria

- The page explains what the studio offers in plain language.
- A visitor can send a contact request from the landing page.
- The MVP works on desktop and mobile.
