# Examples / Ejemplos

This folder shows practical scenarios:

- `new-project-example/`: start from zero.
- `adapt-existing-project-example/`: reverse engineer an existing codebase into specs.
- `002-mcp-end-to-end/`: new-project flow using the local MCP server as the operating layer.

Cada ejemplo contiene idea, primera spec y una bitácora mínima.
