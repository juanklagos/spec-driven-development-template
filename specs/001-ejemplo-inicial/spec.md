# Especificación 001 - Lista de tareas personal

## Historia de usuario principal

Como persona usuaria, quiero crear y marcar tareas como completadas para organizar mis actividades diarias.

## Historias de usuario secundarias

- Como persona usuaria, quiero ver mis tareas pendientes para saber qué me falta.
- Como persona usuaria, quiero eliminar una tarea para mantener limpia mi lista.

## Escenarios de aceptación

1. Dado que estoy en la pantalla principal, cuando escribo una tarea y presiono el botón "Agregar", entonces veo la nueva tarea en la lista de pendientes.
2. Dado que tengo tareas pendientes, cuando marco una tarea como completada, entonces esa tarea cambia a estado completado.
3. Dado que tengo una tarea en la lista, cuando presiono "Eliminar", entonces la tarea desaparece de la lista.

## Requisitos

- Requisito 1: El sistema debe permitir crear tareas con texto no vacío.
- Requisito 2: El sistema debe permitir cambiar el estado de una tarea entre pendiente y completada.
- Requisito 3: El sistema debe permitir eliminar tareas existentes.
- Requisito 4: El sistema debe mostrar separadas las tareas pendientes y las completadas.

## Criterios de éxito

- Criterio 1: Una persona puede crear una tarea en menos de 10 segundos.
- Criterio 2: Una persona puede completar o eliminar una tarea sin recargar la página.
- Criterio 3: El comportamiento es comprensible para una persona nueva sin explicación adicional.
