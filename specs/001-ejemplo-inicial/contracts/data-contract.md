# Contrato de datos - Lista de tareas personal

## Entidad: tarea

Campos recomendados:

- identificador: valor único para distinguir cada tarea.
- texto: contenido de la tarea.
- estado: pendiente o completada.

## Validaciones mínimas

- El campo texto no debe estar vacío.
- El estado debe ser únicamente "pendiente" o "completada".
