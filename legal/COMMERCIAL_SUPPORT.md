# Commercial use and paid support / Uso comercial y soporte de pago

## English

**You do not need to buy anything.** This project is MIT licensed. Commercial use, use inside a
company, embedding it in a paid product, offering it as a hosted service, and using it in an
open-source project are all permitted, free, and require no permission or notice.

There is no commercial license to request. This file exists because an earlier version of the project
required one, and links to it still exist in the wild.

If you want paid help — a workshop for your team, help adapting the workflow to an existing codebase,
or sponsorship of the project — open an issue titled `Support request` and describe what you need.
That is a service arrangement between you and the author. It has nothing to do with permission to use
the software, which you already have.

## Español

**No necesitas comprar nada.** Este proyecto es MIT. El uso comercial, el uso dentro de una empresa,
integrarlo en un producto de pago, ofrecerlo como servicio hospedado y usarlo en un proyecto open
source están todos permitidos, son gratis y no requieren permiso ni aviso.

No hay licencia comercial que solicitar. Este archivo existe porque una versión anterior del proyecto
sí la exigía, y todavía hay enlaces apuntando aquí.

Si quieres ayuda de pago — una formación para tu equipo, apoyo para adaptar el flujo a un código que
ya existe, o patrocinar el proyecto — abre un issue titulado `Support request` y describe qué
necesitas. Eso es un acuerdo de servicios entre tú y el autor. No tiene nada que ver con el permiso
para usar el software, que ya lo tienes.
