# CLAUDE.md - Project Operating Context

This repository uses Spec-Driven Development (SDD) as operating model.

Read first:
- `INSTRUCTIONS.md`
- `AGENTS.md`
- `template-context/README.md`

Hard stop policy:
- No coding until `spec.md` is approved and `plan.md` is consistent.

When blocked by missing approval/consistency:
- Update/refine docs only.
- Report exact gap and next action to unblock.
