# Decisiones de arquitectura

## Propósito

Este documento centraliza decisiones transversales que impactan múltiples specs.

## Plantilla de decisión

| Campo | Descripción |
|---|---|
| Título | Nombre corto de la decisión |
| Contexto | Problema que se resuelve |
| Opciones | Alternativas evaluadas |
| Decisión | Opción elegida |
| Impacto | Consecuencias técnicas y de negocio |

## Ejemplo breve

- Título: Estrategia de autenticación
- Decisión: Mantener sesión con token httpOnly
- Impacto: mejora seguridad en cliente web
